= Workflows
:hardbreaks:
:nofooter:
:icons: font
:linkattrs:
:imagesdir: ./media/

[.lead]
You can use the ONTAP REST API to administer ONTAP systems and perform other storage administrative tasks. Each of the tasks is described using a workflow process consisting of one or more API calls.
